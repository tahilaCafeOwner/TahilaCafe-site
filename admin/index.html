Carousel — Add a Slide

Type the file name you uploaded to assets/images/ and a short alt. Click copy, then paste inside the carousel track in index.html.

Where to paste: in index.html, find the hero carousel and paste this inside <div class="carousel-track"> ... </div> under the other <figure class="slide"> blocks.
